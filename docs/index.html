@magic-examples

below is a collection of the available @magic example pages. these are pages that have been prepared and setup already, and that can be used to start working on a @magic app easily.

  • @magic-examples/minimal demo

    a minimal example for those that want to start from (almost) scratch.

    git repository

  • @magic-examples/example demo

    an example showing client side state management and module usage.

    git repository

  • @magic-examples/project demo

    an example project page, using the @magic-themes/project theme.

    git repository
